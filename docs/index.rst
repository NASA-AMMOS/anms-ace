
Welcome to ACE's documentation!
===============================
This is the AMM CODEC Engine (ACE) for the DTN Management Architecture (DTNMA).
It is part of the larger Asynchronous Network Managment System (ANMS) managed for `NASA AMMOS <https://ammos.nasa.gov/>`_.

The main CLI access to ACE is the :doc:`command <ace_ari>` :command:`ace_ari`, which is also available via the API in :mod:`ace.tools.ace_ari`.

The main API access to ACE is via the :class:`ace.AdmSet`.
